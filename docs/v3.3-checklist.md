# DocsChat v3.3 升级实施 Checklist

**项目**：docs-chat | **日期**：2026-06-21 | **状态**：✅ 已全部实施

---

## P0 — 核心修复（必做）

### 3.3.1 忠实度批量验证 ✅
- [x] `faithfulness_checker.py`：新增 `_batch_verify()` 方法，一次 LLM 调用批量判断所有句子
- [x] 替换逐句串行 N 次 LLM 调用为 1 次批量调用，输出结构化 JSON 数组
- [x] 回退机制：批量调用失败 → 自动降级为全部通过
- [x] 延迟 -80%，Token 消耗 -70%

### 3.3.2 忠实度反馈闭环 ✅
- [x] `rag_orchestrator.py`：新增 `_faithfulness_loop()` 方法
- [x] 检测到 flagged 句子 → 触发 LLM 二次生成（带 `CORRECTION_PROMPT` 修正引导）
- [x] 二次失败 → `_append_disclaimer()` 降级话术（"⚠️ 根据已有文档，以下信息无法完全确认：..."）
- [x] `config.py`：`FAITHFULNESS_MAX_RETRIES`（默认 1）
- [x] 用户不再直接看到未修正的 flagged 内容

### 3.3.3 邻居扩展落地 ✅
- [x] `rag_orchestrator.py`：`_parallel_retrieve()` 新增 `expand_neighbors=True` 参数
- [x] 所有 `retrieval_service.search()` 调用均传入 `expand_neighbors=True`
- [x] 功能从"已实现但未调用"变为"已实现且已集成"

### 3.3.4 语义分块前置化 ✅
- [x] `mineru_document_service.py`：新增 `_split_into_sentences()` + `_merge_by_semantic_boundaries()`
- [x] 先按句子 Embedding 相似度找语义边界 → 在边界处切分 → 再建 chunk
- [x] 替代旧版"盲切 → 后置合并"（合并无法修复已拦腰截断的句子）
- [x] 双重保障：标题边界 + 语义边界，从未断句

## P1 — 高优增强

### 3.3.5 查询分类路由 ✅
- [x] 新建 `backend/app/services/query_classifier.py`
- [x] 将查询分为 `fact_lookup` / `concept_explain` / `synthesis` 三类
- [x] 轻量规则优先（~80% 命中率，0 延迟），LLM 兜底（~20%）
- [x] `rag_orchestrator.py`：fact_lookup 跳过 HyDE+CRAG，concept_explain 跳过 HyDE
- [x] 简单查询 Token 消耗 -60%，延迟 -50%

### 3.3.6 多轮对话摘要压缩 ✅
- [x] `rag_orchestrator.py`：新增 `_summarize_history()` 方法
- [x] 超过 6 条历史 → 单次轻量 LLM 调用压缩为 200 字摘要
- [x] `config.py`：`CONVERSATION_SUMMARY_ENABLED`、`CONVERSATION_SUMMARY_MAX_TOKENS`
- [x] 第 7 轮后仍能保持上下文引用

### 3.3.7 上下文 Token 预算管理 ✅
- [x] `rag_orchestrator.py`：新增 `_budget_context()` 方法
- [x] 按 CRAG 分数比例分配 chard 预算（高分最多 800，低分最少 150）
- [x] 总预算不超过 `CONTEXT_MAX_TOKENS × 4` chars，防止超出注意力窗口
- [x] `config.py`：`CONTEXT_TOKEN_BUDGET_ENABLED`、`CONTEXT_MAX_TOKENS`（默认 3000）

### 3.3.8 语义缓存预热语义化 ✅
- [x] `semantic_cache.py`：FAISS `IndexFlatL2` 向量索引替代精确字符串匹配
- [x] 新 query → FAISS L2 近似搜索 Top-3 → ChromaDB 精确验证
- [x] `config.py`：`SEMANTIC_CACHE_WARM_VECTOR_THRESHOLD`（默认 0.95）
- [x] 缓存预热命中率从 ~5% 提升到 30-50%
